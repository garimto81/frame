1. 영상 파일 선택

분석할 영상 클립(.mov, .mp4 등)을 선택해주세요. (Ctrl 또는 Shift 키를 사용하여 여러 파일을 동시에 선택할 수 있습니다)

2. 기준 설정 (Calibration)

분석할 파일을 선택하면 여기에 영상 목록이 표시됩니다.

목록의 영상을 클릭 후, 미리보기에서 **재생 버튼을 눌러** '선명도 점수'를 확인하고 '흐림 기준값'을 조절하세요.

현재 프레임 선명도: N/A
50

이 값보다 선명도 점수가 낮으면 '흐린 프레임'으로 간주됩니다.

미리보기

3. 전체 영상 분석

기준 설정이 완료되었으면, 선택한 모든 영상의 포커스 이슈를 분석합니다.

참고: 0.5초 간격으로 **가장 중앙에 있는 얼굴**을 찾아 분석합니다. (얼굴 미검출 시 건너뜀)

분석 중... (0/0)

4. 분석 로그

분석을 시작하면 여기에 실시간 로그가 표시됩니다.

5. 분석 결과

분석을 완료하면 여기에 파일별 평균 점수 결과가 표시됩니다.

6. 주요 프레임

'선명'에서 '흐림'으로 전환되는 프레임이 감지되면 여기에 표시됩니다.
